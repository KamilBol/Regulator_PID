# PID Regulator Granulator – KamilBol

Regulator PID na ESP32-S3 DevKitC-1  
Sterowanie falownikiem Optidrive E3 przez Modbus RTU  
Odczyt prądu z PZEM-004T, HMI Nextion 3.5", logi na SD, OTA po WiFi

## Szybki start
1. Zainstaluj VS Code + PlatformIO
2. `git clone https://github.com/KamilBol/PID.git`
3. Otwórz folder w VS Code
4. Build → Upload (pierwszy raz przez USB)
5. Potem tylko OTA po WiFi!

## Struktura kodu
Kod jest podzielony dokładnie tak jak chciałeś – sekcjami z ====== i numerami.
